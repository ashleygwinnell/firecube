Enumeration Type Documentation

enum FireCube::BufferType

Specifies how data is handled.


Buffer type:
STATIC: Data is sent once and stored on the graphics card.
DYNAMIC: Graphics driver will choose the location for the data.
STREAM: Data is sent each time.

enum FireCube::InputMappingType

The type of a mapping.


An action happens upon a key press, while a state is on while the key is down.
